#  教师 AI 教学助手
基于 OpenClaw 开发的教学辅助工具包，帮助高校教师自动批改学生报告、分析成绩数据，**所有数据本地处理，不上传云端**。

---

##  项目简介

本项目为高校教师提供三个开箱即用的 AI 技能（Skills），通过自然语言对话即可完成教学事务处理，无需编程基础。

| 技能名称 | 功能 | 对应用户故事 |
|:---|:---|:---|
|  报告审查器 | 批量检查学生报告是否包含指定章节/知识点，生成缺漏名单 | #4 #5 |
|  成绩分析器 | 自动计算平均分、分数段，生成学情分析简表 | #6 #7 |
|  隐私保护器 | 确保所有数据本地处理，自动拦截网络请求，清理临时文件 | #2 |

---

##  解决的痛点

| 痛点 | 解决方案 |
|:---|:---|
| 逐份阅读学生报告，寻找关键信息耗时耗力 | AI 自动检查，输出缺漏名单 |
| 手动统计成绩，容易出错 | 自动计算平均分、分数段分布 |
| 教学数据涉及隐私，不敢上传云端 | 全部本地运行，不上传任何数据 |
| 教师没有编程基础，害怕使用命令行 | 自然语言对话操作，图形化安装界面 |

##  安装步骤

### 方式一：一键部署包（推荐，零基础教师适用）

1. 下载 `OpenClaw-Windows-一键部署包.zip`
2. 解压到任意文件夹（路径不要包含中文）
3. 双击 `OpenClaw Windows 一键启动.exe`
4. 等待自动安装完成（约 3-5 分钟）
5. 看到「Gateway 在线」即表示安装成功

### 方式二：手动安装（开发者）

```bash
# 1. 安装 OpenClaw
npm install -g openclaw

# 2. 初始化配置
openclaw onboard --install-daemon

# 3. 安装本项目技能
git clone https://github.com/你的用户名/teacher-ai-assistant.git
cp -r teacher-ai-assistant/skills/* ~/.openclaw/workspace/skills/

# 4. 重启 OpenClaw
openclaw gateway restart

# 5. 验证技能安装成功
openclaw skills list

🗂️ 全部技能一览
当前共加载 10 个技能，按类别分：

📚 教学技能（重点介绍）
1. 🎯 grade-analyzer — 成绩统计分析助手
自动计算平均分，划分分数段，生成学情分析简表
适合期中/期末考试后快速了解班级成绩分布
2. 📊 grade-processor — 学生成绩处理器
无乱码处理 Excel/CSV 成绩文件，输出完整统计数据
比 grade-analyzer 更侧重原始数据处理和清洗能力
主打：不乱码、不丢数据、输出完整
3. ✅ report-checker — 高校教师作业批改助手
批量检查学生报告是否包含指定的章节/知识点
适合：收了一堆课程报告/论文，快速筛查哪些学生漏写了关键内容
🔧 系统与工具技能
4. 🔍 find-skills — 技能搜索发现
搜索和发现 OpenClaw 技能市场（ClawdHub 等）的新技能
想装新功能时用这个
5. 🛡️ healthcheck — 安全加固工具
主机安全审计：防火墙、SSH、系统更新检查
用于评估 VPS/服务器安全风险
6. 🔗 node-connect — 节点连接诊断
诊断手机 App（Android/iOS/macOS）与 OpenClaw 配对失败的问题
二维码连接不上、配对 token 失效时的救星
7. 🧪 skill-vetter — 技能审查员
安全审查：在安装第三方技能前检查有没有危险代码
装新技能前的安全检查员
8. 🏗️ skill-creator — 技能创建/编辑工具
创建、编辑、优化、审计技能文件
如果你想自己写一个技能，就靠它
9. 🧵 taskflow — 任务流
用于管理长期、多步骤、可恢复的工作流
适合复杂项目的分解执行
10. 🌤️ weather — 天气查询
通过 wttr.in / Open-Meteo 获取当前天气和预报
无需 API Key
🦞 教学技能的定位
你看，三个教学技能构成了一个完整的教学数据处理管线：

Copy
📄 原始成绩文件 (Excel/CSV)
       ↓
🔄 grade-processor → 清洗、解析、去乱码、完整统计
       ↓
📈 grade-analyzer → 分数段、平均分、学情分析简表
       ↓
📝 report-checker → 批改学生报告，检查知识点覆盖率
