# 教师AI教学助手

基于OpenClaw开发的教学辅助工具，帮助高校教师自动批改学生报告、分析成绩数据。

## 功能特性

- ✅ 报告审查器：批量检查学生报告是否包含指定章节/知识点
- ✅ 成绩分析器：自动计算平均分、分数段分布，生成学情分析简表
- ✅ 隐私保护器：确保所有数据本地处理，不上传云端

## 环境要求

- Windows 10/11
- OpenClaw 2026.4.21+
- Node.js 22+
- Python 3.11+

## 安装步骤

1. 安装OpenClaw一键部署包
2. 将skills文件夹复制到 ~/.openclaw/workspace/skills/
3. 运行 openclaw gateway restart
4. 运行 openclaw skills list 验证
