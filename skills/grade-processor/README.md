# 学生成绩处理器

一个专门处理学生成绩文件的工具，确保无乱码输出完整的成绩统计数据。

## 功能特点

- ✅ **无乱码处理** - 自动检测并处理文件编码问题
- ✅ **多格式支持** - Excel (.xlsx, .xls) 和 CSV 文件
- ✅ **完整统计** - 提供详细的成绩分布分析
- ✅ **简单易用** - 命令行界面，一键分析

## 安装依赖

```bash
pip install pandas openpyxl
```

## 使用方法

### 基本使用
```bash
# 处理Excel文件
python grade_processor.py --file "学生成绩.xlsx"

# 处理CSV文件
python grade_processor.py --file "成绩.csv"

# 指定成绩列名
python grade_processor.py --file "data.xlsx" --column "期末成绩"

# 保存报告到文件
python grade_processor.py --file "成绩.xlsx" --output "分析报告.txt"
```

### 参数说明
- `--file, -f`: 成绩文件路径 (必需)
- `--column, -c`: 成绩列名 (可选，自动检测)
- `--output, -o`: 输出报告文件路径 (可选)

## 支持的文件格式

### Excel 文件
- `.xlsx` (推荐)
- `.xls`

### CSV 文件
支持多种编码：
- UTF-8
- GBK (简体中文)
- GB2312
- UTF-8 with BOM
- Latin-1

## 自动检测功能

1. **自动检测成绩列**：优先查找"成绩"、"score"、"分数"等列名
2. **自动检测编码**：CSV文件自动尝试多种编码
3. **自动处理异常**：跳过无效数据并报告

## 错误处理

- 文件不存在 → 显示错误信息
- 编码错误 → 自动尝试其他编码
- 列名错误 → 显示可用列名列表
- 数据格式错误 → 跳过无效行

## 使用建议

1. **Excel文件**：使用 `.xlsx` 格式，避免使用过时的 `.xls` 格式
2. **CSV文件**：建议使用 UTF-8 编码保存
3. **列名规范**：使用"成绩"、"分数"或"score"作为成绩列名
4. **数据清理**：确保成绩列为纯数字，无文本内容

## 常见问题

### Q: 中文显示乱码怎么办？
A: 工具会自动检测编码，如果仍有问题，请将CSV文件保存为UTF-8编码。

### Q: 找不到成绩列怎么办？
A: 程序会显示所有可用列名，请手动选择正确的列号。

### Q: 文件太大处理慢怎么办？
A: 建议使用Excel格式，CSV大文件处理较慢。

### Q: 如何批量处理多个文件？
A: 可以编写批处理脚本循环调用本工具。
