## 主要功能

1. **无乱码处理** - 自动检测文件编码（UTF-8, GBK, GB2312等）
2. **多格式支持** - Excel (.xlsx, .xls) 和 CSV 文件
3. **完整统计** - 提供详细的成绩数据分析：
   - 平均分、最高分、最低分
   - 中位数、标准差
   - 分数段分布（优秀/良好/中等/及格/不及格）
   - 优秀率、及格率

## 使用方法

### 安装依赖
```bash
pip install pandas openpyxl
```

### 运行命令
```bash
# 基本使用
python grade_processor.py --file "成绩文件.csv"

# 指定成绩列
python grade_processor.py --file "成绩.xlsx" --column "期末成绩"

# 保存报告
python grade_processor.py --file "成绩.csv" --output "分析报告.txt"
```
